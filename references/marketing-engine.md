# 内容优先的心理营销引擎

## 1. 核心原则

营销只放在最后一个Part。前段的心理设计用于提高代入、好奇、理解和信任，不用于提前推销。

默认分配：

```text
Part A 内容钩子与代入：约0%–40%
Part B 解释与认知价值：约40%–75%/85%
Part C 认知植入与解决方案：最后15%–25%，默认不超过30%
```

用户没有明确要求强营销时，营销强度最高为2：

| 强度 | 含义 |
|---|---|
| 0 | 纯内容，没有产品或行动引导 |
| 1 | 自然承接，说明新的判断标准或解决方向 |
| 2 | 克制的解决方案关联，产品、必要机制和结果；默认没有购买CTA |
| 3 | 强销售、优惠、稀缺、反复CTA；默认禁止 |

如果输入没有产品、品牌或商业目标，不得凭空创造营销。Part C改为“最后结论Part”，只完成结果总结、理想状态或自然下一步，营销强度保持0–1。

## 2. 三个Part的职责

### Part A：内容钩子与代入

目标是让观众停留并认出自己。

可使用：异常、反常结果、熟悉场景、身份问题、生活细节、痛点可视化、好奇缺口。

前段允许产品被动存在：可以放在桌面、包内或生活环境里，也可以被人物自然拿起或使用；但不推近、不居中展示、不强调Logo、不解释利益点。

禁止：产品英雄镜头、Logo强调、品牌重复、利益点清单、购买动作、CTA、销售手势和广告腔。

### Part B：解释与认知价值

目标是让观众理解隐藏原因，并接受新的判断标准。

可使用：旧方法失败、表面原因/隐藏原因、机制动画、错误/正确标准、因果链、系统变化和中性证据。

产品可以参与中性功能演示，但不要急着说“我们的产品”。先让答案在逻辑上成立，再在最后Part建立明确的产品关联。

### Part C：认知植入与解决方案

目标不是立刻催单，而是让观众形成“这个产品＝前文解决方案”的认知。默认结果优先：先展示人物或系统获得改善，再让产品获得明确但克制的视觉关联。

产品作为前面结论的具体落点自然进入：

```text
前文结论
→ 需要怎样的解决标准
→ 产品如何满足该标准
→ 一个必要机制或证据
→ 一个可见结果
→ 产品与结果建立记忆关联
```

最后Part不重新制造新痛点，不一次罗列多个利益点，不重复品牌。默认不写购买CTA；只有用户明确要求转化时，才增加一句克制的软行动。

产品英雄镜头是可选项。默认优先使用“人物结果画面＋产品自然可见”；只有需要强化产品识别时，才使用短暂产品主视觉。

非商业内容不强行套用产品结构，最后Part直接完成知识结论或行为建议。

## 3. 人物营销角色

人物不是通用模特。为每个主要人物输出：

| 字段 | 说明 |
|---|---|
| 受众映射 | 代表哪类目标观众 |
| 外部身份 | 年龄、职业、生活状态、使用环境 |
| 内部状态 | 当前焦虑、挫败、渴望或错误信念 |
| 视觉识别物 | 一眼能说明身份的服装、道具、空间 |
| 当前叙事角色 | 观众替身、尝试者、理解者、结果示范者 |
| 情绪弧线 | 开场状态、升级、低点、反转、结束状态 |
| 理想身份 | 最后获得的掌控感、能力或生活状态 |

默认弧线：

```text
Part A：观众替身，问题发生在他身上
→ Part B：尝试者/理解者，经历失败并看见原因
→ Part C：结果示范者，展示获得控制后的状态并建立产品关联
```

人物不得突然面对镜头推销。若需要角色对白，前段说体验与疑问，中段说理解，最后最多自然说一句结果或行动。

## 4. 场景说服任务

每个主要场景只承担一个主要任务：

| 场景 | 主要任务 |
|---|---|
| 日常现实或人物舞台 | 自我识别与代入 |
| 压力累积或问题镜像 | 把抽象痛点变成可见状态 |
| 错误尝试场 | 证明旧路径只能短暂缓解或无效 |
| 原因揭示/HUD | 完成认知反转 |
| 内部或微观空间 | 解释机制与正确标准 |
| 稳定系统 | 展示答案成立后的状态变化 |
| 最后解决方案场景 | 把产品与前文结论对应 |
| 最后结果场景 | 优先展示理想身份，产品自然可见 |
| 可选产品英雄镜头 | 只在需要强化产品识别时短暂使用 |

场景切换必须有语义触发，不因为“需要换画面”而换景。

## 5. 对比设计

每条片选择一条主对比轴，最多两条辅助轴：

- 失控/掌控；
- 表面原因/隐藏原因；
- 短暂刺激/稳定支持；
- 错误路径/正确标准；
- 混乱系统/有序系统；
- 现在的自己/理想中的自己。

把对比转成画面：方向相反、速度快慢、色彩冷暖、构图拥挤/清晰、动作崩塌/稳定、系统暗灭/点亮。不要只靠字幕写Before/After。

## 6. 心理作用分配

### 前段可用

- 异常与好奇：为什么会这样；
- 自我识别：这就是我的生活；
- 共情：角色的反应像真实观众；
- 认知冲突：原来问题不是以为的原因；
- 理解感：复杂原因被清晰可视化。

这些作用必须提供内容价值，不与购买指令绑定。

### 最后Part可用

- 结果想象：观众看见更理想的自己；
- 差异确认：产品满足前文建立的正确标准；
- 机制可信感：展示一个关键过程或证据；
- 认知植入：把产品与正确标准、机制或结果建立关联；
- 可选低摩擦行动：仅在用户要求转化时使用一句自然的下一步。

默认不使用倒计时、稀缺、反复命令、夸张承诺或突然压迫式销售。

## 7. 时长落点

| 总时长 | 认知植入与解决方案Part建议 |
|---|---|
| 30秒 | 最后5–7秒 |
| 45秒 | 最后7–10秒 |
| 60秒 | 最后10–15秒 |
| 75秒 | 最后12–18秒 |

语义完整优先于机械百分比。产品可以在Part A或Part B被动存在或中性使用，但品牌强调、利益点总结、产品英雄镜头和CTA不得提前。用户明确要求产品开场时，才覆盖此默认规则。

内容Part与10秒生成段不要求重合。若Part C从某条10秒视频的中间开始，在导演表中标出精确切换点；最终视频Prompt仍使用镜头编号，不写时间戳。

## 8. 输出与检查

第一阶段增加：

1. 目标受众与当前认知；
2. 人物营销角色与情绪弧线；
3. 三个Part的时间边界；如跨越10秒生成段，标出段内切换点；
4. 每个场景的说服任务；
5. 主对比轴；
6. 每段心理作用与营销强度。

检查：

- 前75%–85%是否真正提供内容价值？
- 前段产品是否仅被动存在或中性使用，没有营销性强调？
- 品牌强调、利益点总结、产品英雄镜头和CTA是否只在最后Part？
- 人物是否先是观众替身，最后才是结果示范者？
- 产品是否由前文逻辑推导出来？
- 最后是否只保留必要机制、一个结果和产品认知关联？
- 未明确要求转化时，是否避免购买CTA？
- 是否默认结果优先，并把产品英雄镜头作为可选项？
- 去掉品牌名后，前两Part是否仍是一条有价值的内容？
- 没有商业目标时，是否避免凭空创造产品和营销？
