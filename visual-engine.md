# 已验证模式库

只有用户明确要求“学习这个风格”“加入Skill”或同义指令时才更新本文件。记录视觉风格与镜头模式，避免把两者混为一谈。

## 默认视觉风格：欧美半写实3D动画电影

### 视觉指纹

- 竖版9:16；
- 欧美3D动画电影角色，略夸张比例与半写实材质；
- 细腻皮肤、眼睛、眉毛、毛发和服装材质；
- 明亮蓝色渐变摄影棚背景，中心亮、边缘略深；
- 柔和正面主光、轻微顶部轮廓光、柔和地面接触阴影；
- 人物居中，周围允许信息图式图标和道具；
- 黄色与白色粗体字幕通常位于腰部到胸部，但字幕后期添加；
- 发光轮廓、HUD、粒子、扫描、失焦和运动模糊保持同一3D体系。

### 一致性风险

- 极端微距变成真人摄影；
- 图标变成二维贴纸；
- 产品或道具使用不同光线；
- 全身与中景的人物脸型漂移；
- 蓝色背景的曝光和色温跨段变化。

## 镜头模式1：压力累积—极端特写—尝试—崩塌

### 适用

问题钩子、疲惫、焦虑、信息过载、环境压力、错误解决方式。

### 10秒骨架

| 时间 | 变化 | 通道 |
|---|---|---|
| 0–0.6s | 异常人物姿态抬头 | 人物 |
| 0.6–1.2s | 手势与视线改变 | 人物 |
| 1.2–2.2s | 道具分批进入 | 道具 |
| 2.2–3.4s | 状态恶化、镜头推进 | 人物＋镜头 |
| 3.4–4.2s | 轮廓、乌云或系统出现 | 特效＋道具 |
| 4.2–5.1s | 快速推进到极端微距 | 镜头＋尺度 |
| 5.1–6.2s | 硬切回中景，执行错误尝试 | 人物＋道具 |
| 6.2–7.3s | 短暂改善后崩塌 | 人物＋状态特效 |
| 7.3–8.4s | 更多道具进入 | 道具＋构图 |
| 8.4–8.7s | 失焦再对焦 | 特效 |
| 8.7–10s | 全面包围并保持微动 | 构图＋道具 |

### 关键原则

- 极端微距是景别中断，不是风格中断；
- 道具分批进入，不在首帧全部出现；
- 尝试必须产生可见动作和短暂结果；
- 崩塌与开场状态呼应，但信息密度更高。

## 镜头模式2：视觉论证链

### 适用

30–75秒产品解释、教育、健康、科技、效率、心理和商业内容。

### 结构

```text
问题可视化
→ 理想与现实对比
→ 主角主动对抗
→ 证明外部方法无效
→ 产品或答案进入
→ 进入内部空间
→ 解释底层机制
→ 错误方案与正确方案对比
→ 状态回顾
→ 产品英雄镜头
```

### 视觉章节

一个论证阶段对应一个主要视觉空间：人物舞台、信息图、HUD、动作场、产品舞台、内部通道、微观系统、稳定系统、英雄镜头。每6–12秒迁移空间，使用主体、道具、主色或运动方向桥接。

## 镜头语法

### 渐进式视觉累积

**适用：** 展示压力、风险、输入、选择或证据增加。

每0.3–0.8秒加入一个新物体。改变方向、速度、前后景和材质音，不同时进入。最后形成密集构图。

### 动词动作化

**适用：** 口播包含躲避、阻挡、抓住、追赶、过滤、推动、击碎等动作词。

人物必须真正执行动词。动作同时改变人物姿势、主体位置、道具轨迹和构图重心。

### 语义转场

**适用：** 口播出现inside、outside、underneath、but、instead、crash、disappear、rebuild等词。

把词义变成空间、尺度或状态转场，不随机插入炫技镜头。

### 桥接物体

**适用：** 从一个视觉空间迁移到另一个空间。

保留一个物体、轮廓、颜色或运动方向。例如产品瓶→胶囊→体内胶囊→矿物粒子。

### 尺度跳变

**适用：** 解释从环境影响到人物、器官、微观机制或产品结果的因果链。

在环境级、人物级、器官级、微观级和产品英雄级之间切换，避免连续同尺度静态讲解。

### 抽象系统可视化

**适用：** 无法直接拍摄的健康、心理、效率、科技和商业概念。

把抽象状态转成电池、插槽、电路、仪表、齿轮、节点、流量或网络。必须让系统发生可读变化：空→填入、暗→点亮、不稳→稳定、堵塞→流动。
